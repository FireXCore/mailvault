# Code of conduct

## Standard

Project participation must remain professional, technical and respectful.

Acceptable conduct includes:

- discussing code and design decisions directly;
- providing reproducible evidence;
- reviewing contributions on technical merit;
- respecting confidentiality and privacy;
- acknowledging uncertainty and correcting mistakes.

Unacceptable conduct includes:

- harassment or discriminatory behavior;
- publishing private information;
- submitting real credentials or confidential mailbox content;
- personal attacks;
- intentionally misleading security or compatibility claims;
- disruption of project infrastructure or discussions.

## Enforcement

Report conduct concerns privately to the repository maintainers. Maintainers may edit or remove content, close discussions, restrict participation or block accounts when necessary to protect the project and its contributors.
