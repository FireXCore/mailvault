# Standards and implementation references

MailVault is designed against public standards and official implementation guidance.

## Email and IMAP

- RFC 5322, Internet Message Format: https://www.rfc-editor.org/rfc/rfc5322
- RFC 9051, IMAP4rev2: https://www.rfc-editor.org/rfc/rfc9051
- RFC 6154, IMAP LIST Special-Use: https://www.rfc-editor.org/rfc/rfc6154
- RFC 7162, CONDSTORE and QRESYNC: https://www.rfc-editor.org/rfc/rfc7162
- RFC 8474, IMAP OBJECTID: https://www.rfc-editor.org/rfc/rfc8474
- Gmail IMAP extensions: https://developers.google.com/workspace/gmail/imap/imap-extensions

- GitHub repository best practices: https://docs.github.com/en/repositories/creating-and-managing-repositories/best-practices-for-repositories
