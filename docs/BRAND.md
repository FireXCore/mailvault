# Brand assets

MailVault uses a restrained technical identity intended for documentation, package indexes and repository presentation.

## Assets

- `assets/logo.svg`: square project mark.
- `assets/banner.svg`: README and documentation header.
- `assets/social-preview.png`: GitHub repository social preview.
- `assets/architecture.svg`: architecture overview.
- `assets/evidence-model.svg`: canonical evidence relationship diagram.
- `assets/doctor-output.svg`: documented CLI example.
- `assets/sync-output.svg`: documented sync example.

## Colors

```text
Navy       #0B172A
Blue       #1E3A5F
Steel      #5B708A
Teal       #2B7A78
```

Do not add decorative emoji, mascots, gradients, clip art or informal badges to official project documentation.
